## 1. 产品概述

### 1.1 产品背景

Struct Parser VS Code 插件是一款专为嵌入式开发、底层驱动开发、硬件寄存器调试场景设计的可视化工具。它能够将十六进制数据按照 C 语言结构体定义进行解析，以树形结构展示各字段的数值、二进制表示和十六进制值。

### 1.2 产品定位

- **目标用户**: 嵌入式工程师、驱动开发者、硬件调试工程师
- **核心价值**: 提升寄存器/内存数据解析效率，降低二进制数据分析门槛
- **使用场景**: 调试硬件寄存器、分析内存转储、验证数据结构

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| Struct | C 语言结构体，一种复合数据类型 |
| Union | C 语言联合体，共享内存的复合数据类型 |
| Field | 结构体/联合体的成员字段 |
| Offset | 字段在结构体中的位偏移量 |
| LSB | Least Significant Bit，最低有效位 |
| MSB | Most Significant Bit，最高有效位 |

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 JSON 数据导入 (已实现)

**需求描述**: 支持导入 struct-parser 生成的 JSON 文件，加载结构体定义。

**功能细节**:
- 支持通过文件选择对话框导入 JSON 文件
- 自动保存导入路径到 VS Code 配置
- 支持从配置路径自动加载
- 解析失败时给出明确错误提示

**验收标准**:
- [x] 点击 "Import JSON File" 按钮打开文件选择器
- [x] 成功加载后显示结构体数量
- [x] 加载失败时显示错误信息
- [x] 自动保存路径到 `structParser.jsonPath` 配置

#### 2.1.2 Hex 值解析 (已实现)

**需求描述**: 将输入的十六进制值按照选中的结构体定义进行解析。

**功能细节**:
- 支持带或不带 `0x` 前缀的 hex 值
- 自动处理数据位宽（小于结构体大小时补零，大于时截断）
- 支持大端序（MSB-first）数据解析
- 解析结果实时展示

**验收标准**:
- [x] 输入 `1234ABCD` 或 `0x1234ABCD` 均可解析
- [x] 数据位宽小于结构体时低位补零
- [x] 数据位宽大于结构体时低位截断
- [x] 解析结果包含：字段名、类型、位宽、二进制值、十进制值、十六进制值

#### 2.1.3 树形控件展示 (已实现)

**需求描述**: 以树形结构展示解析后的字段数据，支持嵌套结构体/联合体。

**功能细节**:
- 支持展开/折叠嵌套结构
- 显示字段的完整信息（名称、类型、位宽、值）
- 支持字段值的编辑
- 支持字段搜索和高亮

**验收标准**:
- [x] 点击箭头可展开/折叠子结构
- [x] 显示字段的十进制值、十六进制值
- [x] 嵌套结构体/联合体正确展示层级关系
- [x] 字段值编辑后重新计算
- [x] 字段搜索过滤

#### 2.1.4 侧边栏 Struct Explorer (已实现)

**需求描述**: 在 VS Code Activity Bar 中常驻显示结构体列表。

**功能细节**:
- 分类展示 Structs 和 Unions
- 显示每个结构体的位宽
- 点击结构体自动打开解析器并选中
- 工具栏支持导入和刷新

**验收标准**:
- [x] 在 Activity Bar 显示 Struct Parser 图标
- [x] 分类展示 Structs 和 Unions
- [x] 显示结构体位宽
- [x] 点击结构体打开解析器
- [x] 工具栏支持 Import 和 Refresh

## 3. 非功能需求

### 3.1 性能需求

| 指标 | 要求 |
|------|------|
| 解析响应时间 | < 100ms（100 个字段以内） |
| 界面加载时间 | < 500ms |
| 内存占用 | < 100MB |
| 支持的最大结构体 | 1000 个字段 |

### 3.2 兼容性需求

| 项目 | 要求 |
|------|------|
| VS Code 版本 | >= 1.74.0 |
| 操作系统 | Windows, macOS, Linux |
| 浏览器内核 | Chromium (VS Code 内置) |

### 3.3 可靠性需求

- 解析错误时给出明确提示，不崩溃
- 无效 JSON 文件时优雅处理
- 异常 hex 输入时给出错误信息

### 3.4 安全性需求

- 不执行用户输入的任意代码
- 不访问网络（本地插件）
- 敏感数据不离开本地
