# Struct Parser VS Code Extension

A VS Code extension for visualizing C-style struct data from hex values. Works with the output from [struct-parser](https://github.com/NeeWe/struct-parser).

## Features

- **Hex Value Parsing**: Parse hex values (e.g., `0x1234ABCD`) into struct fields
- **Visual Field Breakdown**: See each field's binary representation, decimal value, and hex value
- **Struct Browser**: Tree view of all structs and their fields
- **Context Menu Integration**: Right-click any hex value to parse it

## Requirements

- VS Code 1.74.0 or later
- A struct parser JSON output file (generated by [struct-parser](https://github.com/NeeWe/struct-parser))

## Installation

### From Source

```bash
cd struct-parser-vscode
npm install
npm run compile
```

Then press `F5` to open a new Extension Development Host window.

### From VSIX (after packaging)

```bash
vsce package
# Install the generated .vsix file in VS Code
```

## Configuration

Set the path to your struct parser JSON output in VS Code settings:

```json
{
  "structParser.jsonPath": "/path/to/your/output.json"
}
```

Or the extension will automatically look for `output.json` or `structs.json` in the workspace root.

## Usage

### Open the Struct Parser Viewer

1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Type "Open Struct Parser Viewer"
3. Select the command

### Parse a Hex Value

**Method 1: Using the Viewer**
1. Open the Struct Parser Viewer
2. Enter a hex value (e.g., `0x1234ABCD` or `1234ABCD`)
3. Select a struct from the dropdown
4. Click "Parse"

**Method 2: From Editor Selection**
1. Select a hex value in any editor
2. Right-click and select "Parse Hex Value"
3. The viewer will open with the value pre-filled

### View Struct Definitions

1. Open the Explorer sidebar
2. Look for the "Struct List" view
3. Expand structs to see their fields

## Example

Given this struct definition:

```c
struct ControlReg {
    uint1  enable;      // 1 bit
    uint1  interrupt;   // 1 bit
    uint2  mode;        // 2 bits
    uint4  reserved;    // 4 bits
    uint8  prescale;    // 8 bits
    uint16 timeout;     // 16 bits
};
```

And hex value: `0xABCD1234`

The extension will display:

| Field | Type | Bits | Binary | Value | Hex |
|-------|------|------|--------|-------|-----|
| enable | uint1 | 1 | 1 | 1 | 0x1 |
| interrupt | uint1 | 1 | 0 | 0 | 0x0 |
| mode | uint2 | 2 | 10 | 2 | 0x2 |
| ... | ... | ... | ... | ... | ... |
